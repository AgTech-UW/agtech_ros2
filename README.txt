# AgTech Computing: ROS 2 Development Environment

## Prerequisites
1. Install **Docker Desktop**
2. Install **VS Code**
3. Install **"Dev Containers"** extension in VS Code

## How to Start
1. Clone this repository.
2. Open the folder in VS Code.
3. A popup will appear: "Reopen in Container". Click it.
   - *If you missed it, press F1 and type "Dev Containers: Reopen in Container".*
4. Wait for the build (takes ~5-10 mins the first time).
5. Once the terminal opens, type `ros2` to verify installation.

## Common Commands
- Build workspace: `colcon build`
- Source environment: `source install/setup.bash`
